How It Works

1
Upload
Upload your raw DNA file from 23andMe, AncestryDNA, MyHeritage, or VCF format.
⚙️
2
Parse & Normalize
We parse and normalize variants across formats, extracting rsIDs and genotypes.
🔗
3
Annotate
Variants are annotated via ClinVar, Ensembl VEP, MyVariant.info, and gnomAD.
🧠

Analyze
Our engine categorizes variants and computes polygenic risk scores.
📊
5
Report
Download your report as an interactive web dashboard, PDF, or JSON.

Supported File Formats

23andMe
.txt
Tab-separated file with rsid, chromosome, position, genotype columns. Download from Browse Raw Data.
AncestryDNA
.txt
Tab-separated with rsid, chromosome, position, allele1, allele2. Download from Settings → Download DNA Data.
MyHeritage
.csv
Comma-separated with RSID, CHROMOSOME, POSITION, RESULT columns.
VCF
.vcf
Standard Variant Call Format. Supports both GRCh37 and GRCh38 reference assemblies.
